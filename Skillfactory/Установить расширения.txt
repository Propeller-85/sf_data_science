Для установки библиотеки введите в командную строку (или командную строку Anaconda) следующее:

Matplotlib
pip install matplotlib

Plotly:
pip install plotly

nbformat:
pip install nbformat
(Остальное может не понадобиться. На всякий случай записал.
Method 1. reinstall ipykernel via: pipenv install ipykernel
Method 2. pip install --upgrade nbformat)
